# 遗忘角落探险家小程序

校园失物招领微信小程序前端演示版，适合直接导入微信开发者工具，也适合整理后上传到 GitHub。

## 功能

- 首页：失物/招领信息流、快捷筛选、分类筛选、关键词搜索、统计信息、图片缩略图。
- 发布：寻物/招领切换、表单校验、分类选择、发布身份、图片选择/预览/删除、写入本地存储。
- 详情：物品详情、图片预览、认领核验、联系对方、标记完成、完成状态防重复。
- 消息：系统通知、联系请求、未读数量、单条已读、全部已读、清空消息。
- 我的：学生认证、守护积分、发布记录、进行中/完成统计、最近发布、重置演示数据。

## 运行

1. 打开微信开发者工具。
2. 导入此目录。
3. 如需公开仓库演示，保留当前 `touristappid`；如需真机调试，替换为你自己的 AppID。
4. 点击编译即可。

## 上传 GitHub 前建议

- 不要提交 `project.private.config.json`、`.agents/`、`.codex/` 这类本地环境文件，已加入 `.gitignore`。
- 保留 `README.md`、`LICENSE` 和 `project.config.json`，方便别人导入和了解项目。
- 如果你要长期维护，后续可以再补 `CHANGELOG.md` 和 GitHub Issues 模板。

## 数据方式

当前版本使用 `wx.setStorageSync` 模拟数据库，可以完整体验前端交互流程。
